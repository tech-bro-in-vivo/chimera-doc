# 快应用组件

# 视图容器

名称 功能说明
movable-view 在页面中可以拖动滑动的视图容器
cover-image 能够覆盖在原生组件之上的图片组件
cover-view 能够覆盖在原生组件之上的文本组件
movable-area movable-view的可移动区域
scroll-view 可滚动的视图区域
swiper 滑块视图容器
swiper-item 仅可放置在swiper组件中的子组件，宽高默认为 100%
view 视图容器

# 原生组件说明

名称 功能说明
native-component ## 原生组件

快应用中的部分组件是由客户端创建的原生组件，这些组件有：

  • canvas
  • input（仅在 focus 时表现为原生组件）
  • map
  • textarea
  • video

# 原生组件的使用限制

原生组件脱离在 WebView 渲染流程外，需要留意以下限制：

  • 原生组件的层级是最高的，页面中的其他组件无论设置 z-index 为多少，都无法盖在原生组件上
