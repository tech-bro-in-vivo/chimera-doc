# qa.getWxPaymentType()

获取当前可用的微信支付调用方式

微信支付详细使用方式请参考微信支付说明

# 参数

none

# 返回值

返回值 备注
APP 微信 app 调用方式，服务端向微信支付下单时，trade_type 需要是 APP，参考微信 app 支付
MWEB 微信网页调用方式，服务端向微信支付下单时，trade_type 需要是 MWEB，参考微信网页支付
none 微信未安装

注意：在 getType 返回值为 APP 时，开发者将只能使用 app 方式，不能使用网页方式。

# 示例代码

console.log(qa.getWxPaymentType())
