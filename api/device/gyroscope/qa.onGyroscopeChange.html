# qa.onGyroscopeChange(function callback)

注册陀螺仪数据回调监听器

需要先通过 qa.startGyroscope() 方法启动陀螺仪监听，并传入 interval 参数做为频率。

可以使用 qa.stopGyroscope() 方法停止监听。

# 参数

# function callback

陀螺仪数据的回调

# 参数

# Object res
属性 类型 说明
res Object

res 的结构

属性 类型 说明
x number x 轴的角速度
y number y 轴的角速度
z number z 轴的角速度
