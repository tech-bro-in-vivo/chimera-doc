# qa.makePhoneCall(Object object)

拨打电话

# 参数

# Object object

属性 类型
默认值
必填
 说明
phoneNumber string 是 需要拨打的电话号码
success function 否 接口调用成功的回调函数
fail function 否 接口调用失败的回调函数
complete function 否 接口调用结束的回调函数（调用成功、失败都会执行）

# 示例代码

qa.makePhoneCall({
  phoneNumber: '1340000' //仅为示例，并非真实的电话号码
})
