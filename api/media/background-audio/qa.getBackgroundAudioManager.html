# BackgroundAudioManager qa.getBackgroundAudioManager()

获取全局唯一的背景音频管理器。 快应用切入后台，如果音频处于播放状态，可以继续播放。但是后台状态不能通过调用 API 操纵音频的播放状态。

若需要在快应用切后台后继续播放音频，需要在 app.json 中配置 requiredBackgroundModes 属性。开发版和体验版上可以直接生效，正式版还需通过审核。
