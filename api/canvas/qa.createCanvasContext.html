# CanvasContext qa.createCanvasContext(string canvasId, Object this)

创建 canvas 的绘图上下文 CanvasContext 对象

# 参数

# string canvasId

要获取上下文的 canvas 组件 canvas-id 属性

# Object this

在自定义组件下，当前组件实例的 this，表示在这个自定义组件下查找拥有 canvas-id 的 canvas ，如果省略则不在任何自定义组件内查找
