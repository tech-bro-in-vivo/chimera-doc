# 开始

# 安装开发工具

开始项目前，请先安装快应用开发工具（IDE），下载地址见相关下载

IDE 使用说明可以参考 IDE 文档

# 创建项目

在开发前，需要了解快应用的的代码是由下面四种类型的文件构成：

  • .json 后缀的配置文件
  • .qxml 后缀的模板文件
  • .css 后缀的样式文件
  • .js 后缀的脚本逻辑文件

使用快应用开发工具可以非常方便的新建项目。
首先第一步，需要在开发工具中安装两个插件hap-chimera-project(模版插件)和hap-chimera-toolkit(调试工具插件)，

安装完毕之后选择 新建工程 -> 选择模板 【chimera-demo】 即可创建一个默认的快应用项目。

新建完一个工程之后，可以利用 IDE 进行开发、预览和调试。编写代码时，保存文件 IDE 会重新编译项目并刷新页面。

  • 点击图中 1 处 icon 可以打开预览界面
  • 点击图中 4 处 icon 可以打开二维码，用于真机上扫码下载

# 真机预览调试

真机上预览需要先在手机上安装以下 apk :

  • 快应用引擎: app-release.apk
  • 快应用调试器: quickapp_debugger.apk
  • webview 内核: V5WebView.apk

真机上可以有下面两种方式预览 rpk：

# 1. 扫码安装

确保手机和电脑处于同一个网段。然后再点击快应用调试器上的【扫码安装】，扫描 IDE 上的二维码即可下载预览项目。

# 2. 本地安装

本地安装不要求手机和电脑处于同一个网段。只需要将项目编译生成的 output/**.rpk 文件推送到手机，然后点击快应用调试器上的【本地安装】，选择 rpk 文件就可以预览了。
