# 快应用(Light)是什么？

快应用是基于手机硬件平台的新型应用形态，旨在与手机系统深度整合，为用户提供更加场景化的体验。它具备传统 APP 完整的应用体验，无需安装、即点即用；覆盖 10 亿设备，与操作系统深度集成，探索新型应用场景。

快应用特点之一在于：它基于前端技术栈，相比于原生，拥有更高的开发效率；但为了追求更好的应用体验，快应用采取基于 Native 原生渲染（各类小程序则采用 Webview 渲染方式），且提供的是一套与当前流行前端框架有所差异的语法；这就使得在开发快应用时候，有额外的学习成本，且项目代码不易与 Web、小程序等相互复用；为此，vivo、华为等厂商，有采取相同的标准，推出快应用(Light 版)（为叙述方便，下文统称为「快应用」），它同小程序一样，采取 Webview 渲染方式，使得开发者没有额外学习成本（有 Web、小程序开发经验即可），能够使用 Web 上各种能力，同时令用户有良好应用访问体验。与此同时，官方有提供一键转换小程序为快应用工具，进一步大幅降低小程序迁移至快应用成本。
