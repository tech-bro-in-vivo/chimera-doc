# 小程序转快应用light版

对于已经有微信小程序的开发者来说，我们提供了一个更为方便快捷的工具可以直接将开发者的微信小程序项目转换为快应用light，只需要简单的一条命令就可以实现，🍺Cheers！

# 使用方法

  • 使用 npm install qa-adapter -g 或者 yarn add -g qa-adapter 安装快应用转换器。
  • 安装成功之后 用命令 adt [sourceDir] [destDir?optional] 来转换项目。

# 参数说明

参数 作用
-v，--version 输出工具版本号
-l，--logs 在转换后项目的根目录中输出转换记录文件logs.txt
