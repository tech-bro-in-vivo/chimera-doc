# 渲染层

框架的渲染层由 QXML 与 CSS 编写，对应页面和自定义组件。

逻辑层的数据通过渲染层的界面展现，同时渲染层接收交互事件，发送回逻辑层处理。

QXML 用于描述页面的结构。

QJS 是可运行在渲染层的特殊 js。

CSS 用于描述页面的样式。

组件是视图的基本组成单元。

这模块将介绍渲染层的事件、基础组件、动画、样式等
