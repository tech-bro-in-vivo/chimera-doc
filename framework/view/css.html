    # 样式

    快应用样式和网页 CSS 样式大体相同。需要留意以下差异：

    1. 快应用样式新增了对 rpx 单位的支持，可以根据屏幕宽度进行自适应
    2. 快应用中，定义在 app.css 中的样式为全局样式，所有页面和自定义组件内都可以使用；定义在页面的样式在该页面及页面内组件都可用；定义在自定义组件中的样式为局部样式，仅该组件可用
