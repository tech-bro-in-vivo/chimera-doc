    # 基础组件

    快应用框架提供了丰富的组件，开发者可以在 qxml 中直接使用。可用组件列表

    注意：组件与属性都需要小写，以连字符-连接

    # 公共属性

    所有组件都有以下属性：

    属性名 类型 描述 注解
    id String 组件的唯一标示 需要保持页面内唯一
    class String 组件的样式类 在对应的 CSS 中定义
    style String 组件的内联样式 内联样式，优先级高于样式类
    hidden Boolean 组件是否显示 所有组件默认显示
    data-* Any 自定义属性 组件上触发的事件时，会发送给事件处理函数
    bind* / catch* EventHandler 组件的事件 详见事件

    # 独有属性

    不同的组件有各自特有的属性，详情参考各个组件文档。
