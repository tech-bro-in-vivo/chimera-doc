# QJS 语法参考

QJS 是快应用自定义脚本语言，运行在渲染层，配合 QXML 一起构建页面。

QJS 是 JavaScript 语言的子集，与 JavaScript 是不同的语言，其语法并不与 JavaScript 一致。
