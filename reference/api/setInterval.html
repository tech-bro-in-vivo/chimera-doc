                                          # number setInterval(function callback, number delay, any rest)

                                          设定定时器。每隔一定的周期执行回调函数

                                          # 参数

                                          # function callback

                                          回调函数

                                          # number delay

                                          时间间隔，单位 ms。

                                          # any rest

                                          param1, param2, ..., paramN 等附加参数，它们会作为参数传递给回调函数。

                                          # 返回值

                                          定时器 ID。这个值可以传递给 clearInterval 来取消该定时。
